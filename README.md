# OctoDiary-py
Неофициальная Python-библиотека для использования API МЭШ / Моя Школа.

## Установка

### Новейшая версия:
``` bash
$ pip install https://github.com/OctoDiary/OctoDiary-py/archive/main.zip --upgrade --force-reinstall
```

### Стабильная версия:
``` bash
$ pip install octodiary --upgrade
```
